### Mushroom Classification System 
**Tech:** Python, pandas, scikit-learn, Streamlit

Developed a machine learning system to classify mushrooms as edible or poisonous using supervised learning techniques. The project involved full data preprocessing, model evaluation, and deployment through an interactive web application.

**Key Features:**
- Performed exploratory data analysis (EDA) and handled missing data
- Implemented multiple models (KNN, Decision Tree, Naive Bayes)
- Evaluated performance using 10-fold cross-validation
- Built a complete ML pipeline and exported the best model
- Developed a Streamlit web app for real-time predictions

GitHub: https://github.com/hjadoon21/Mushroom-Classification/tree/main
